# Ghost-Astro

    Ghost-Astro is the Astro version of Ghost!

    # Features

    Ghost has a lot of features including

    * Cloaking
    * Theming
    * Lots of Games
    * And A Proxy!

    # Deploying

    ```
    git clone https://github.com/The-Ghost-Network/Ghost-Astro
    cd Ghost-Astro
    npm i
    npm run build
    npm start
    ```

    Deploying using pm2

    ```
    npm install -g pm2
    git clone https://github.com/The-Ghost-Network/Ghost-Astro
    cd Ghost-Astro
    npm i
    npm run build
    pm2 start run-server.js
    ```
